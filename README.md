# Task 5: SQL Joins

## Objective
To learn how to combine data from multiple tables using different types of SQL joins.

## Tools Used
- MySQL Workbench
- MySQL Server

## What I Did
- Created two tables: Customers and Orders.
- Inserted sample data into both tables.
- Performed the following SQL joins:
  - INNER JOIN
  - LEFT JOIN
  - RIGHT JOIN
  - FULL OUTER JOIN (using UNION)
  - CROSS JOIN
  - SELF JOIN
- Saved all queries in `task5.sql`.
- Added screenshots of query outputs.

## Key Concepts Covered
- Relationship between tables.
- Using INNER, LEFT, RIGHT, and FULL joins.
- How FULL OUTER JOIN is implemented in MySQL using UNION.
- Understanding CROSS JOIN and Cartesian product.
- Using SELF JOIN to compare rows within the same table.
